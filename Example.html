3. Custome some default settings:

3.1 do something before submit and after submit:

As default, after submit button being clicked, the form will be validate one more again and submit to server when validation completed as true. But you can meddle to this process by use function beforeSubmit and afterSubmit. Easy provide you 2 function "beforeSubmit()" and "afterSubmit()" to do some thing before and after form get submitted.

Example:

name (only "easy" accepted):
email :

form: <form action="example.php" method="post" id="form2" class="easyValidate[ajax(),beforeSubmit(),aftersubmit()]">

JS code: 
        jQuery(document).ready(function () {
                $(#formID1).easySetting.beforeSubmit = function () {
                        if ($(#formID1).isFormValidated())
                                alert("hello, this is alert before form submit");
                };
                $(#formID1).easySetting.afterSubmit = function () {
                                alert("hello, this is alert after form submitted, the result of form after validate is
"+$("#d").isFormValidated());
                };
        });

3.4 Enable/disable auto scroll:

set scroll:0 to disable scroll and scroll:1 to enable scroll

Example: <form action="example.php" method="post" id="e" class="easyValidate[beforeSubmit(),scroll:false]">
